# Keyboard Shortcuts

Productivity tips and keyboard shortcuts for Investment Portfolio Manager.

## Overview

Currently, the Investment Portfolio Manager primarily uses mouse/touch navigation. This page documents any available keyboard shortcuts and productivity tips.

## Browser Shortcuts

These standard browser shortcuts work in the application:

### Navigation

| Shortcut | Action |
|----------|--------|
| `Ctrl/Cmd + R` | Refresh page |
| `Ctrl/Cmd + Shift + R` | Hard refresh (clear cache) |
| `Ctrl/Cmd + W` | Close tab |
| `Ctrl/Cmd + T` | New tab |
| `Backspace` or `Alt + ‚Üê` | Browser back |

### Page Operations

| Shortcut | Action |
|----------|--------|
| `Ctrl/Cmd + F` | Find on page |
| `Ctrl/Cmd + +` | Zoom in |
| `Ctrl/Cmd + -` | Zoom out |
| `Ctrl/Cmd + 0` | Reset zoom |
| `Ctrl/Cmd + P` | Print page |

### Forms and Input

| Shortcut | Action |
|----------|--------|
| `Tab` | Next field |
| `Shift + Tab` | Previous field |
| `Enter` | Submit form |
| `Esc` | Close modal/dialog |
| `Ctrl/Cmd + A` | Select all text |
| `Ctrl/Cmd + C` | Copy |
| `Ctrl/Cmd + V` | Paste |

## Application Tips

### Quick Navigation

**Using the Address Bar:**
- Bookmark frequently used pages
- `/tickers` - Go directly to Tickers
- `/portfolio` - Go directly to Portfolio
- `/research` - Go directly to Research
- `/help` - Go directly to Help

**Example:** `https://yourapp.com/tickers`

### Form Efficiency

When adding positions:
1. Use `Tab` to move between fields
2. Use `Enter` to submit (on some forms)
3. Use `Esc` to cancel/close dialogs

### Search and Filter

On pages with search/filter:
- Click search box
- Type ticker or keyword
- `Enter` to search

## Productivity Tips

### Multiple Tabs

Open multiple pages in tabs:
- Tab 1: Tickers (monitoring)
- Tab 2: Research (news)
- Tab 3: Key Metrics (analysis)

### Browser Bookmarks

Bookmark common workflows:
- `/tickers` - Daily portfolio check
- `/research` - Market news
- `/help?section=guides/api-setup` - API troubleshooting

### Quick Data Entry

For frequent position updates:
1. Keep a text file with your common tickers
2. Copy-paste into the add position form
3. Use `Tab` to move through fields quickly

## Accessibility

### Screen Readers

The application works with screen readers:
- Navigate using standard screen reader commands
- Form fields are labeled
- Buttons have descriptive text

### Tab Navigation

You can navigate the entire app using only the keyboard:
1. Use `Tab` to move through interactive elements
2. Use `Enter` or `Space` to activate buttons/links
3. Use arrow keys in dropdowns and selects

### Zoom

For better visibility:
- Zoom in: `Ctrl/Cmd + +`
- Zoom out: `Ctrl/Cmd + -`
- Reset: `Ctrl/Cmd + 0`

## Developer Console

For troubleshooting:
- `F12` - Open developer tools
- `Ctrl/Cmd + Shift + I` - Open developer tools
- `Ctrl/Cmd + Shift + J` - Open console tab
