# First-Time Setup

Detailed guide for setting up Investment Portfolio Manager for the first time.

## Overview

This guide provides comprehensive instructions for configuring the application after creating your account. For a quicker overview, see the [Quick Start Guide](quick-start-guide.md).

## Account Creation

### Step 1: Navigate to the Application

Visit the Investment Portfolio Manager application URL.

### Step 2: Create Your Account

1. Click "Create Account" on the sign-in screen
2. Enter your email address
3. Create a strong password (minimum requirements will be displayed)
4. Confirm your password
5. Click "Create Account"

### Step 3: Verify Your Email

1. Check your email inbox for a verification code
2. Enter the verification code in the application
3. Complete the verification process

### Step 4: Sign In

Use your email and password to sign in for the first time.

## Initial Configuration

### Configure API Access

To access real-time market data, you'll need a Financial Modeling Prep API key. See the detailed [API Setup Guide](../guides/api-setup.md) for complete instructions.

**Quick steps:**
1. Visit [financialmodelingprep.com](https://financialmodelingprep.com/)
2. Create a free account
3. Copy your API key
4. In the app, go to Settings
5. Paste your API key and save

See the [Portfolio Organization Guide](../guides/portfolio-management.md) for details on creating and managing portfolios.
