# Portfolio Visualization

Interactive 3D visualization of your investment portfolio.

## Overview

The Portfolio page (also called Flexible Portfolio) provides a visual representation of your investments using an interactive 3D globe visualization.

## Features

### 3D Globe Visualization

An interactive 3D globe scene displays your portfolio visually:
- Navigate and rotate the view
- Visual representation of holdings
- Immersive portfolio overview

### Portfolio Filtering

Filter the visualization by portfolio:
- Select specific portfolios to view
- Focus on individual accounts or strategies
- Compare portfolio allocations visually

### Today's Performance

Monitor your portfolio's daily performance:
- Total portfolio value
- Today's dollar change
- Today's percentage change
- Color-coded gains/losses

## How to Use

1. Navigate to the Portfolios page from the top navigation
2. Use the portfolio filter dropdown to select a portfolio
3. Interact with the 3D visualization
4. View performance metrics at the top

## Tips

- Use this page for a high-level portfolio overview
- Switch to Tickers page for detailed position management
- Filter by portfolio to analyze specific accounts
