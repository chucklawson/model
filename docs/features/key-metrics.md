# Key Metrics

Analyze company financial statements and key performance metrics.

## Overview

The Key Metrics page allows you to:
- View company financial statements
- Compare quarterly vs annual data
- Analyze key ratios and metrics
- Track performance over time

## How to Use

### Search for a Company

1. Navigate to Key Metrics page
2. Enter a stock ticker in the search box
3. Click "Search" or press Enter

### View Financial Data

The page displays:
- **Income Statement** - Revenue, expenses, net income
- **Balance Sheet** - Assets, liabilities, equity
- **Cash Flow** - Operating, investing, financing cash flows
- **Key Ratios** - P/E, P/B, ROE, ROA, debt ratios

### Toggle Views

Switch between:
- **Quarterly Data** - Last 4 quarters
- **Annual Data** - Last 3-5 years

## Key Metrics Explained

### Valuation Metrics
- **P/E Ratio** - Price to earnings
- **P/B Ratio** - Price to book value
- **P/S Ratio** - Price to sales

### Profitability Metrics
- **ROE** - Return on equity
- **ROA** - Return on assets
- **Profit Margin** - Net income / revenue

### Financial Health
- **Debt to Equity** - Total debt / shareholders equity
- **Current Ratio** - Current assets / current liabilities
- **Quick Ratio** - (Current assets - inventory) / current liabilities

## Analysis Tips

1. **Compare to Industry Peers** - Metrics are more meaningful in context
2. **Look for Trends** - Are metrics improving or declining?
3. **Consider Multiple Metrics** - Don't rely on a single ratio
4. **Quarterly vs Annual** - Check for seasonality
